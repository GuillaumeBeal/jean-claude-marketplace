# Formats de citation professionnels

## Citation française standard

### Textes législatifs et réglementaires
```
Loi n° 2016-1691 du 9 décembre 2016 relative à la transparence, à la lutte contre la corruption et à la modernisation de la vie économique (dite « loi Sapin 2 »)
Décret n° 2025-968 du 23 septembre 2025 relatif à [...]
Ordonnance n° 2016-131 du 10 février 2016 portant réforme du droit des contrats
```

### Articles de code
```
Art. 1240 du Code civil
Art. L. 225-1 du Code de commerce
Art. R. 4121-1 du Code du travail
```
- Toujours indiquer l'état : « en vigueur » ou « abrogé depuis le [date] »
- Si modifié : « dans sa rédaction issue de [texte modificateur] »
- Si version historique : « dans sa rédaction en vigueur au [date] »

### Jurisprudence judiciaire (Cour de cassation)
```
Cass. [formation], [date], n° [pourvoi], [publication]
```
Exemples :
```
Cass. com., 5 mars 2026, n° 24-15.820, publié au Bull.
Cass. civ. 3e, 12 février 2026, n° 24-18.788, inédit
Cass. soc., 15 janvier 2025, n° 23-12.456, publié au Bull.
Cass. crim., 8 décembre 2025, n° 24-80.123
Cass. ass. plén., 3 juillet 2025, n° 23-18.789, publié au Bull.
```

Formations :
| Abréviation | Formation |
|---|---|
| `civ. 1re` | Première chambre civile |
| `civ. 2e` | Deuxième chambre civile |
| `civ. 3e` | Troisième chambre civile |
| `com.` | Chambre commerciale |
| `soc.` | Chambre sociale |
| `crim.` | Chambre criminelle |
| `ass. plén.` | Assemblée plénière |
| `ch. mixte` | Chambre mixte |

### Jurisprudence administrative
```
CE, [formation], [date], n° [numéro], [recueil]
```
Exemples :
```
CE, ass., 12 avril 2025, n° 456789, Lebon
CE, sect., 5 février 2026, n° 467890, Lebon
CE, 3e et 8e ch. réun., 18 mars 2025, n° 458123
TA Paris, 15 décembre 2025, n° 2300123/2-1
CAA Paris, 2e ch., 7 novembre 2025, n° 24PA01234
```

### Conseil constitutionnel
```
Cons. const., [date], n° [numéro], [type]
```
Exemples :
```
Cons. const., 23 mars 2017, n° 2017-750 DC
Cons. const., 12 octobre 2025, n° 2025-1098 QPC
```

### Conventions collectives
```
Convention collective nationale [intitulé], IDCC [numéro]
Art. [numéro] de la convention collective nationale [intitulé], IDCC [numéro]
```
Exemple :
```
Art. 19 de la convention collective nationale de la métallurgie, IDCC 3248
Convention collective nationale des bureaux d'études techniques (dite « Syntec »), IDCC 1486
```

### Délibérations CNIL
```
CNIL, délibération n° [numéro] du [date], [objet]
```
Exemple :
```
CNIL, délibération n° SAN-2024-020 du 5 décembre 2024, Société XYZ
```

### Circulaires
```
Circ. [ministère], [date], NOR : [NOR]
```

### Journal Officiel
```
JORF n° [numéro] du [date], texte n° [numéro]
```

## Format international (pour memos cross-border)

Quand le contexte est international (due diligence M&A, avis cross-border), ajouter une traduction en anglais du titre et un résumé en une ligne :

```
Art. 1240 of the French Civil Code (« Code civil »)
[Provides that any act causing damage to another obliges the person at fault to compensate the victim — general tort liability provision]

French Law No. 2016-1691 of 9 December 2016 on transparency, the fight against corruption and the modernisation of economic life (known as "Sapin 2 Law")
```

## Identifiants Légifrance à inclure

Toujours mentionner l'identifiant Légifrance entre parenthèses pour permettre un accès direct :
```
Art. 1240 du Code civil (LEGIARTI000032041571)
Cass. com., 5 mars 2026, n° 24-15.820 (JURITEXT000053679063)
```

## Règles générales

1. **Toujours vérifier l'état en vigueur** avant de citer
2. **Date de consultation** : mentionner « consulté le [date] via Légifrance »
3. **Articles abrogés** : citer l'ancien ET le nouveau avec concordance
4. **Modifications récentes** : signaler la date et le texte de la dernière modification
5. **Ne jamais citer de mémoire** — toujours vérifier via le MCP
